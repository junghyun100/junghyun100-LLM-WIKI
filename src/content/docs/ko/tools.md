# 도구 & 템플릿

위키 운영을 돕는 도구와 템플릿 모음입니다.

## CLI 도구

- `ingest.sh` — 소스 파일 추가 및 메타데이터 생성
- `lint.py` — 위키 건강도 점검 (모순, 고아 페이지, 메타데이터 누락)
- `index-gen.py` — index.md 자동 갱신

## 프롬프트 템플릿

- `ingest.md` — Ingest 작업용 시스템 프롬프트
- `query.md` — Query 작업용 시스템 프롬프트
- `lint.md` — Lint 작업용 시스템 프롬프트

## 평가 가이드

- RAG 평가 체크리스트
- 모델 비교 평가 기준
- 벤치마크 실행 가이드
